КУДА КЛАСТЬ ФОТОГРАФИИ

Фоновые снимки — прямо сюда, с такими именами:

  cover.jpg        фон заставки на первом экране
  about-bg.jpg     фон раздела «О себе»
  portrait.jpg     портрет автора, пропорция 3:4
  contact.jpg      фото в блоке «Связаться»

Расширение любое: jpg, png, webp, avif.

Альбомы — в папку albums, по папке на альбом. Подробности в albums/README.txt.

ПОДГОТОВКА
  - обложки альбомов режьте под 21:9, ширина до 2000 px
  - фон заставки 1920x1080, он всё равно затемняется
  - формат WebP вдвое легче JPEG при том же качестве
  - держите вес одного файла до 300 КБ

Ничего прописывать в коде не нужно: при пуше на GitHub список
собирается автоматически.
